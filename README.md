# ☕ CoffeeShop CMS

A modern coffee shop website with admin dashboard built using React, TypeScript, and Tailwind CSS.

## 🚀 Features

- ✨ Modern UI with Framer Motion animations
- 🛒 Real-time cart system with drawer
- 🔥 Admin dashboard (CRUD menu)
- 💾 Persistent data using LocalStorage
- 📱 Fully responsive design
- 🎨 Premium UI polish & micro-interactions

## 🛠 Tech Stack

- React + TypeScript
- Tailwind CSS
- Framer Motion
- Context API
- Lucide Icons
- Vite

## 📦 Installation

```bash
pnpm install
pnpm dev
```
